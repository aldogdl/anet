Política de Privacidad

Uso de Datos

Nuestra aplicación solo utiliza la API de WhatsApp para responder automáticamente a los mensajes de servicio. No solicitamos ni almacenamos datos personales de los usuarios.

Finalidad

La aplicación está diseñada exclusivamente para integraciones B2B, proporcionando respuestas automáticas a consultas de servicio. No recopilamos ni utilizamos datos de los usuarios para otros fines.

Seguridad

Nos comprometemos a mantener la confidencialidad y seguridad de los datos que puedan transitar a través de nuestra aplicación, limitándonos a lo estrictamente necesario para el funcionamiento del servicio.
